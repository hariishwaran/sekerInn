# Sekar Inn — Website

A single-page website for Sekar Inn, an affordable roadside stay in Arumuganeri, Thoothukudi, Tamil Nadu.

## Features
- Hero section with parallax scrolling and scroll-linked animations
- Room listings (AC / Non-AC, suites, dorms, driver rooms) with 3D hover tilt
- Auto-scrolling horizontal guest reviews
- Contact details and click-to-call phone number
- Custom Sekar Inn logo and favicon
- Fully responsive, with reduced-motion support for accessibility
